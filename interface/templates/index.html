Install SecureLock

Chrome did not expose the automatic install prompt. You can still install SecureLock manually:

  1. Open the Chrome menu (three dots) in the top-right.
  2. Choose Install SecureLock (or Add to desktop / Add to Home screen on mobile).
  3. Follow the on-screen steps to add the app.

On mobile Android, use the browser menu → Add to Home screen.
